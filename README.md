# SnakeNladder on SwiftBot

SnakeNladder is my first-year university robotics project. The aim was to design and implement the classic Snake and Ladder game on the university-provided SwiftBot platform.

The project creates the game board and uses console output as the main user interface. `Sharing.java` is a helper class for sharing the SwiftBot API across the project, while `SwiftBotMovement.java` controls how the robot moves according to the Snake and Ladder board for the SwiftBot player.

This version was completed in 2025 and reorganized into a cleaner folder structure after moving from Eclipse to VS Code.
